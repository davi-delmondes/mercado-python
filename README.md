# 🛒 Mercado Python

Sistema de mercado desenvolvido em Python para trabalho do curso.

## 📌 Sobre o projeto

O **Mercado Python** é um sistema simples executado pelo terminal, criado para simular o funcionamento básico de um mercado.

O projeto permite o cadastro de usuários, login como administrador ou cliente, cadastro de produtos, controle de estoque, carrinho de compras e relatório de vendas.

## ⚙️ Funcionalidades

- Cadastro de usuários
- Login de administrador e cliente
- Cadastro de produtos
- Listagem de produtos
- Edição de preço e estoque
- Carrinho de compras
- Remoção de itens do carrinho
- Finalização de compra
- Relatório de vendas
- Salvamento de dados em JSON

## 🧰 Tecnologias utilizadas

- Python
- JSON
